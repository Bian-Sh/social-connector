Social Connector
================

1つのAPIでTwitter/Facebookにポスト出来るUnityプラグインです。

2012/12/08現在 iOS版のみ

```
///=================
/// Twitter
///=================
		
SocialConector.PostMessage (SocialConector.ServiceType.Twitter,"text");


///=================
/// Facebook
///=================

SocialConector.PostMessage (SocialConector.ServiceType.Facebook, "text")	
	
```
